### Rodando local

Pimeiro inicie o root

- `cd root && npm install`
- `npm start`

Inicie o auth

- `cd auth && npm install`
- `npm start`

Inicie a navbar

- `cd navbar && npm install`
- `npm start`

Inicie o footer

- `cd footer && npm install`
- `npm start`

Inicie o login

- `cd login && npm install`
- `npm start`

Inicie o partners

- `cd partners && npm install`
- `npm start`


Inicie a api fake

- `cd partners && npm run json:server`
